# sarahaglund.github.io

Editorial minimal luxury showcase for Sara Haglund.

## Quick Edit Guide (GitHub UI)

This site is intentionally built with plain HTML/CSS/JS so content can be updated without local setup.

## File structure

- index.html: Home and featured sections
- about.html: Profile and process section
- portfolio.html: Fashion design gallery
- content.html: Content creation gallery
- styles.css: Shared visual style
- site.js: Reveal animation and media fallbacks
- media/images: Add image files here
- media/videos: Add MP4 files here

## Replace images and videos

1. Upload files into media/images or media/videos in GitHub.
2. Open the target HTML file.
3. Find lines near comments that say Replace image path below or Replace video path below.
4. Change only the src value.

Examples:

- Image: media/images/portfolio-look-01.jpg
- Video: media/videos/content-reel-01.mp4

## Recommended naming

- Images: project-name-01.jpg
- Videos: project-name-01.mp4
- Posters (optional): project-name-01-poster.jpg

## If media is not uploaded yet

The design will show an intentional placeholder card instead of a broken-looking block.

## ChatGPT helper prompt

Use this in ChatGPT when you want to update content:

"I have a static website with files index.html, about.html, portfolio.html and content.html. Help me update only text copy and media src values while preserving the existing HTML structure and classes."
